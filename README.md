**TextUtils** is a premium, modern text manipulation utility built for developers, writers, and content creators. It offers a distraction-free, immersive typing experience with powerful text transformation tools, real-time statistics, and a sophisticated aesthetic.

## Key Features

*   **Immersive Typing Experience**: Features a custom typewriter sound engine (Web Audio API) with distinct feedback for typing and deleting, paired with the classic `Courier Prime` font.
*   **Premium UI/UX**: Designed with a "Pitch Black" dark mode for OLED screens and a crisp, clean light mode. The interface utilizes glassmorphism, smooth `framer-motion` animations, and a responsive layout.
*   **Text Transformations**: Comprehensive suite of tools including case conversion (camelCase, snake_case, etc.), text cleaning (trim, remove duplicates), and formatting.
*   **Real-time Analytics**: Instant feedback on word count, character count, reading time, and more.
*   **SEO Optimized**: Fully configured with Open Graph and Twitter Card metadata for optimal social sharing.
*   **Progressive Loading**: Features a custom animated loading bar with a seamless overlay transition.

## Tech Stack

*   **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
*   **Animations**: [Framer Motion](https://www.framer.com/motion/)
*   **Audio**: Web Audio API (Custom Oscillators)
*   **Font**: Google Fonts (Courier Prime, Inter)

## Getting Started

### Prerequisites

Ensure you have Node.js installed.

### Installation & Build

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to view the application.
